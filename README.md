# AI vs Human Text Detection

This project implements an AI vs Human-generated text detection system
using TF-IDF + linguistic features and a Random Forest classifier.

 Note:
This code is designed to run on **Google Colab** and uses Google Drive
for storage. No local execution setup is provided.

## Model
- Random Forest
- TF-IDF (unigrams + bigrams)
- Linguistic features (POS ratios, readability)

## Platform
- Google Colab
